# outreach

Email and outreach content for DoDAO's robotics services.

This folder holds the **text** of outreach emails — not the sending
pipeline. Sending is handled elsewhere (the `outreach-data` agent in
the parent Discord-bot project owns the send pipeline).

## Contents

- [`email-templates.md`](email-templates.md) — Four cold-outreach email
  variants for senior robotics-company contacts (CEO, co-CEO, co-founder,
  VP). Two dynamic placeholders: `[name]` and `[company name]`.

## Source material the email content is drawn from

- The robotics services page: https://dodao.io/robotics
- The two-service poster (Simulation Setup + Synthetic Data) shared
  alongside this task.

## Writing rules (apply to anything added here)

1. Simple English. Short sentences.
2. No marketing language — no superlatives, no buzzwords, no vague
   promises.
3. Professional and direct. State who, what, and the next step.
4. Use placeholders for fields that change per recipient. Keep the
   number of placeholders small — every extra placeholder is one more
   thing that can be wrong in a send.
