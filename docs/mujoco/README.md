# MuJoCo

A small set of beginner-friendly notes about **MuJoCo** — what it is,
when it is the right tool, and how it compares to Gazebo and NVIDIA
Isaac Sim.

These notes sit alongside the broader simulation overview in
[`../generic-workflow/03-software-stack/07-simulation.md`](../generic-workflow/03-software-stack/07-simulation.md).
That file lists every common simulator in one place. This folder zooms
in on MuJoCo only.

## Read in order

1. **[01-what-is-mujoco.md](01-what-is-mujoco.md)** — What MuJoCo is,
   the kinds of work people use it for, how it is different from Gazebo
   and Isaac Sim, and why a beginner should learn it.

More files will be added here over time (install on Ubuntu, port a URDF
to MJCF, run MJX on a GPU, sim-to-real on the myCobot, etc.). They will
all follow the same plain-English style.
